# Ayura OS

Ayura OS (formerly Vaidyo) is a modern, AI-native hospital operating system built on Next.js, Supabase, and Expo.

This folder contains the core planning documents and technical specifications for the 2026 rebrand and rebuild.

### Contents:
- **`PRD.md`** - Product Vision, Epics (including AI Scribe, Copilot, Pharmacy)
- **`SPRINTS.md`** - 8-Sprint roadmap
- **`STORIES.md`** - Gherkin BDD stories
- **`ARCHITECTURE.md`** - Supabase + App Router + AI Orchestrator blueprint
- **`COMPONENT_MAPPING.md`** - UI/UX execution via Shadcn & Tremor
